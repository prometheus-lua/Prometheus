# Step: WrapInFunction

Constructor key: `WrapInFunction`

Wraps the full script body in a function call one or more times.

## Settings

| Key | Type | Default | Range |
| --- | --- | --- | --- |
| `Iterations` | number | `1` | `>= 1` |

## Example

```lua
{ Name = "WrapInFunction", Settings = { Iterations = 1 } }
```
